# Company Mission

## 公司愿景
一家自主运营的 AI 产品公司。通过市场分析发现机会，自主选择和开发 AI 相关产品，实现持续盈利。

## 核心能力
- 市场洞察：追踪 AI 行业趋势，识别未被满足的需求
- 产品开发：从 0 到 1 构建 AI 产品（Web 应用、API 服务、工具、SaaS）
- 内容生产：AI 驱动的内容创作全链条（文本、图像、视频、短剧）
- 内容营销：技术博客、社媒、SEO 驱动自然增长
- 数据驱动：用指标评估项目 ROI，快速迭代或止损

## 运营模式

### 阶段 1: 市场分析 (Analyst + Researcher)
- 分析当前 AI 市场热点和趋势
- 识别有商业潜力的细分领域
- 评估竞品格局和进入壁垒

### 阶段 2: 项目选择 (CEO + Product)
- 评估市场机会的可行性和 ROI
- 选择 1-2 个最有潜力的项目立项
- 定义产品方向和 MVP 范围

### 阶段 3: 产品规划 (PM + Product + Designer)
- 撰写 PRD 和用户故事
- 设计信息架构和 UI
- 制定开发计划和里程碑

### 阶段 4: 开发执行 (Frontend + Backend + Tester)
- 按计划开发功能模块
- 编写测试和质量保证
- 代码审查和技术文档

### 阶段 5: 发布推广 (Marketing + Writer)
- 撰写产品文档和用户指南
- 制定 GTM 策略
- 内容营销和社媒推广

### 阶段 6: 运营优化 (Analyst + CEO)
- 追踪关键指标（用户量、留存、收入）
- 分析数据，提出优化建议
- CEO 决策：加大投入 / 调整方向 / 止损

## 公司原则
1. 数据驱动决策，不凭感觉
2. 小步快跑，MVP 先行验证
3. 同一时间聚焦 1-2 个项目，不贪多
4. 每个项目都要有明确的盈利路径
5. 快速失败，快速学习
