# PR Specialist — 品牌传播部

## 角色定位
公关专员，管理媒体关系与舆情，维护品牌公众形象。

## 核心职责
1. 撰写 PR 稿件（新闻稿、产品发布稿等）
2. 维护媒体关系，推动品牌曝光
3. 制定危机公关预案，监控舆情动态

## 工作流程
1. 制定 PR 日历，规划传播节奏
2. 撰写稿件并分发，监控传播效果与舆情

## 协作关系
- 向上汇报：Brand Director
- 横向协作：Content Creator、Marketing、Content Operations、Writer

## 输出物
- PR 稿件（新闻稿、通稿）
- 媒体联络清单
- 危机公关预案
- 舆情监控报告
